Пояснение к работе:

Программа представляет собой простой будильник с интерфейсом в командной строке.

Настройка будильника:

Пользователь вводит время будильника в формате 24-часового времени (HH MM).
Программа ожидает наступления указанного времени.
Звуковое и визуальное уведомление:

Когда наступает установленное время будильника, программа выводит уведомление о том, что пора просыпаться.
Звуковой сигнал воспроизводится в цикле с заданной длительностью (dur), используя инструкции ассемблера.
После звукового сигнала отображается меню управления уведомлениями.
Управление уведомлениями:

Пользователю предлагается выбрать один из вариантов:
1: Изменить длительность звукового сигнала (увеличивается в 10 раз при каждом выборе).
2: Изменить цвет уведомления (цвет увеличивается с каждым выбором).
3: Завершить программу.
В случае некорректного ввода, программа просит пользователя ввести корректное значение.
Цветовое выделение:

При выборе изменения цвета уведомления, вся область экрана циклически окрашивается в новый цвет (color).
Выход из программы:

При выборе завершения программы, программа завершается, и на экран выводится сообщение о завершении.
Дополнительные функции:

Программа использует инструкции ассемблера для управления цветом и звуковым сигналом.
Есть задержка (delay), используемая для создания паузы между звуковыми сигналами.
